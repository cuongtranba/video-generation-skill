## Goal

Derive the pipeline board model — per-step state and cost, the active step, failures, the retry command map, the cost cap, formatted event rows, and media URLs — as pure functions with no React.

## Purpose

Owns the frontend's pipeline derivation layer: deriveSteps folds ProjectState plus the raw event log (and client-side in-flight flags) into the six ordered StepInfo entries; activeStep picks the step needing attention; lastFailure/stepClearedBy track un-superseded RunFailed stages; retryCommandFor maps a stage to its command; capUsd reads the cost cap; formatEvent maps a VidgenEvent to a log row; mediaUrl maps an event asset path to the api's /media route. Non-goal: renders nothing and touches no store — components consume these functions.
