## Goal

Route POST /api/commands/* to command handlers, serve read endpoints from Postgres, handle media asset uploads, and serve the SPA and media files.

## Parent Fit

| Field | Value |
| --- | --- |
| Container | c3-10 api |
| Category | feature |
| Boundary | Node.js HTTP server; exposed on PORT (default 8080) |
| Status | active |

## Purpose

Owns createHttpServer, COMMAND_HANDLERS dispatch map (8 commands), parseTuneInput, handleUploadAsset (multipart/form-data, 100 MB limit), handleListAssets, listProjects (Postgres projection read), getProject (Postgres projection read), and serveStatic (SPA + media). Per-process idempotencyCache keyed on client-supplied idempotencyKey. Non-goal: does not own command business logic — it parses and delegates.
