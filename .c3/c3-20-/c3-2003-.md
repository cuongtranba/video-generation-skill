## Goal

Provide FPT.AI async-polling TTS and ElevenLabs TTS behind a TTSProvider interface; NewFromConfig selects the implementation from config.yaml.

## Parent Fit

| Field | Value |
| --- | --- |
| Container | c3-20 worker |
| Category | feature |
| Boundary | In-process Go; calls external FPT.AI and ElevenLabs HTTP APIs; writes MP3 to shared media volume |
| Status | active |

## Purpose

Owns TTSProvider interface, FPTAIProvider (async-polling: POST → poll URL until 200), ElevenLabsProvider, and NewFromConfig factory. FPT.AI is the active provider; ElevenLabs is implemented but not yet default. Non-goal: does not own the Voice/Speed domain types — those are in domain/.
