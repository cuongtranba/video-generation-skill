## Goal

Select which vendor implements each pipeline category and validate that the required secrets exist before any run.

## Parent Fit

| Field | Value |
| --- | --- |
| Parent | c3-1 vidgen CLI process |
| Layer | Foundation — read at startup by cli before wiring providers |
| Depends on | gopkg.in/yaml.v3, Go stdlib; .env and ~/.vidgen/config.yaml |
| Consumed by | c3-106 cli, and every category factory (NewFromConfig) |

## Purpose

Owns `ProvidersConfig` (which provider per category, from `config.yaml`) and `Config` (API keys from `.env`), plus `LoadProviders`, `DefaultProvidersConfig`, and `ValidateForProviders` which fails fast when a selected provider's key is missing. Non-goals: never holds secrets in the YAML, never constructs providers itself.
